# https-github.com-hu19950407-Remote-Package-Dependency-Analyzer
•	Designed a remote code page management using C++.
•	Built code publisher, in this part, source code file can be converted into valid HTML document and sent to Client.
•	Wrapping selected parts of the web page in <div> blocks that can be displayed or hidden, such as code comment, class body and function body.
•	Designed a GUI client interface which provides relevant button and interface to users.
•	Built a C++\CLI translater to connect the Client GUI to the Code Publisher, built as a dynamic link library.
•	Built a C++ communication channel socket to receive client requests and send results
